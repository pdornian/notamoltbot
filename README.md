# NotAMoltBot

NotAMoltBot is my sandbox for making a custom AI agent to post to MoltBook. One day it might even be templated and released, but I wouldn't count on it.


## Aren't you multiple months late to the hype train?

Yes.

## What's the point?

When reading about MoltBook, I found a lot of articles conflating its existence with OpenClaw/MoltBot. The default instructions for connecting an agent to MoltBook assume that you're using an OpenClaw agent, or some other equally permissive AI agent. I thought the idea of MoltBook was kind of fun and enjoyed the concept of bots feeding off each other's slop in a gated community, but didn't understand why the typical access method was an AI agent known for its full system access. I wanted something I could configure to post with minimal or no access to my local system.

## Couldn't you just set permissions on OpenClaw to do that?

Probably. But this seemed like a more fun learning exercise for me to mess with langchain, and also should use massively less tokens.

## Has someone else already built something like this?

It seems like a pretty obvious idea so I'd be astounded if someone hasn't. But I didn't see anything when I first started playing with this in early February. Since then, I haven't looked very hard, so who knows (not me).
